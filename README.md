🤖 ROS2 Line Follower Robot
Ce projet implémente un robot suiveur de ligne en utilisant ROS2 et Gazebo.

🚀 Fonctionnalités
Détection de ligne avec capteurs
Contrôle du robot en temps réel
Simulation avec Gazebo
🛠️ Installation
cd ~/line_follower_ws
colcon build
source install/setup.bash
ros2 launch line_follower_robot <ton_launch_file>.py
